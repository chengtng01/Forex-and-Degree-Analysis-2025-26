# Forex-and-Degree-Analysis-2025-26
University Coursework: Supervised and Unsupervised Learning

This repository contains the R programming scripts used to generate the models, statistical tests, and visualisations for my Economics and Finance coursework.

Project A (Unsupervised Learning): Contains the K-Means clustering algorithm applied to the Singapore Ministry of Education's GES 2022 dataset to identify hidden economic tiers among university degrees.

Project B (Supervised Learning): Contains the machine learning pipeline (Multiple Linear Regression, Logistic Regression, Decision Trees, and Random Forests) used to predict hourly volatility and price direction for the EUR/USD Forex market.

Note: The datasets used in this analysis are publicly available via the Singapore Government Data Portal and Kaggle.
